## はじめに

AE AI Extensionは、After Effectsでのキャラクターアニメーション制作を効率化する拡張機能です。AIとの自然言語対話により、複雑なアニメーション操作をシンプルに実現できます。

### 主な特徴

- 🗣️ 自然言語でアニメーション指示
- 🔄 リアルタイムプレビュー
- 📁 プリセット保存・再利用
- 🔀 複数キャラクター対応
- 📴 オフライン作業対応

---

## 初期設定

### 1. 拡張機能の起動

After Effectsで:
1. **ウィンドウ** > **エクステンション** > **AE AI Extension**

### 2. ログイン

1. メールアドレスとパスワードでログイン
2. 新規ユーザーは「アカウント作成」から登録

### 3. 設定確認

設定アイコンから以下を確認:
- AI モデル選択（GPT-4 / Claude）
- 言語設定
- 自動保存の有無

## キャラクター作成

### 手順

1. **新規キャラクター**ボタンをクリック

2. **キャラクター名**を入力
   - 例: "キャラクターA", "主人公"

3. **レイヤーマッピング**を設定

   After Effectsのレイヤー名とパーツを対応付け:

   | パーツ | レイヤー名例 |
   |-------|------------|
   | 頭 | head, 頭 |
   | 体 | body, 胴体 |
   | 右腕 | arm_R, 右腕 |
   | 左腕 | arm_L, 左腕 |
   | 右足 | leg_R, 右足 |
   | 左足 | leg_L, 左足 |

4. **保存**をクリック

### レイヤー命名規則

推奨されるレイヤー命名規則:

```
キャラクター名_パーツ名_サブパーツ
例: charA_arm_R_upper  (キャラクターAの右腕上部)
```

## スケルトン設定

スケルトンはキャラクターのボーン構造を定義します。

### ボーンの追加

1. **スケルトン**タブを開く
2. **ボーン追加**をクリック
3. 以下を設定:
   - **名前**: ボーン名（例: spine, arm_R）
   - **親ボーン**: 接続先の親ボーン
   - **位置**: [X, Y] 座標
   - **回転制限**: 最小〜最大角度

### 推奨ボーン構造

```
spine (背骨)
├── head (頭)
├── arm_R (右腕)
│   ├── forearm_R (右前腕)
│   └── hand_R (右手)
├── arm_L (左腕)
│   ├── forearm_L (左前腕)
│   └── hand_L (左手)
├── leg_R (右足)
│   └── foot_R (右足首)
└── leg_L (左足)
    └── foot_L (左足首)
```

### 回転制限の設定

パーツごとの推奨制限値:

| パーツ | 最小角度 | 最大角度 |
|-------|---------|---------|
| 頭 | -30° | 30° |
| 腕 | -180° | 180° |
| 前腕 | -150° | 0° |
| 足 | -45° | 90° |

## AIチャット操作

### 基本的な使い方

チャット入力欄に自然言語でアニメーション指示を入力:

```
「手を振るアニメーションを追加」
```

AIがJSXコードを生成し、自動的に適用されます。

### 効果的なプロンプト例

#### アクション指示

```
「キャラクターに歩くアニメーションを追加」
「ジャンプして着地する動きを作成」
「手を上げて挨拶する動作」
```

#### パラメータ指定

```
「2秒間の手を振るアニメーション」
「ゆっくりとした頷きを3回繰り返す」
「スピード1.5倍で走る動き」
```

#### 感情表現

```
「驚いた表情にする」
「嬉しそうに跳ねる」
「悲しげに俯く」
```

#### 複合指示

```
「歩きながら手を振る」
「ジャンプして、着地時に少し屈む」
```

### レスポンスの確認

AIの回答には以下が含まれます:

1. **生成されたコード** - JSXコード（自動適用）
2. **説明** - 何が行われたかの説明
3. **プレビュー** - After Effectsでの変更

## プリセット管理

### プリセットの保存

1. アニメーションを生成
2. **プリセットとして保存**をクリック
3. 以下を入力:
   - **名前**: プリセット名
   - **カテゴリ**: gesture, walk, emotion など
   - **タグ**: 検索用タグ

### プリセットの適用

1. **プリセット**タブを開く
2. プリセットを検索・フィルタ
3. 適用したいプリセットをクリック
4. **適用**ボタン

### プリセットのカスタマイズ

既存プリセットをベースに調整:

```
「基本歩行プリセットをベースに、もっと元気よく」
```

## スタイル設定

### スタイルとは

アニメーション全体に適用される視覚的な特性:
- イージング（加速・減速のカーブ）
- タイミング
- 振幅の大きさ

### スタイルの作成

1. **スタイル**タブを開く
2. **新規スタイル**をクリック
3. パラメータを設定:
   - **イージング**: linear, easeIn, easeOut, easeInOut, bounce
   - **振幅スケール**: 動きの大きさ（0.5〜2.0）
   - **速度スケール**: 全体の速度（0.5〜2.0）

### スタイルの適用

チャットで指定:

```
「コミカルスタイルで走る」
「リアルスタイルの歩行」
```

## Tips & ベストプラクティス

### 効率的なワークフロー

1. **まずスケルトンを完成させる**
   - ボーン構造が正しくないと、アニメーションが期待通りに動作しません

2. **シンプルな指示から始める**
   - 複雑な動きは、シンプルな動きを組み合わせて作成

3. **プリセットを活用する**
   - よく使う動きはプリセット化して再利用

### よくある問題と解決策

#### 「レイヤーが見つかりません」

→ レイヤーマッピングを確認し、After Effectsのレイヤー名と一致させる

#### 「動きが不自然」

→ スケルトンの回転制限を調整する

#### 「アニメーションが速すぎる/遅すぎる」

→ プロンプトで速度を指定:「ゆっくり」「速く」「2秒かけて」

### キーボードショートカット

| ショートカット | 機能 |
|--------------|------|
| `Ctrl/Cmd + Enter` | メッセージ送信 |
| `Ctrl/Cmd + Z` | 最後の操作を取り消し |
| `Ctrl/Cmd + S` | 現在の状態を保存 |
| `Esc` | パネルを閉じる |
